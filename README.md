## Prerequisites

This project requires NPM. Check if you have it:

```bash
$ npm -v
```

## Installation

Start with cloning this repo on your local machine:

```bash
$ git clone https://github.com/mylenavendramini/react-apps.git
$ cd react-apps
```

## Usage

```bash
$ npm start
```
